# 🌍 InsightGeo — LULC Change Analysis Dashboard

An interactive geospatial dashboard designed to analyze **Land Use / Land Cover (LULC) changes** over time and transform complex spatial data into clear, actionable insights.

## 🧠 Problem

Understanding how land evolves over time (e.g., agriculture turning into urban areas) is essential for **urban planning, environmental monitoring, and decision-making**.

However, raw geospatial datasets are:

* Difficult to interpret
* Not accessible to non-technical users
* Lacking clear, actionable insights

---

## 💡 Solution

InsightGeo simplifies this by providing an **interactive, user-friendly dashboard** that enables users to:

* Explore land-use changes across different years
* Identify key transitions (e.g., *Agriculture → Built-up*)
* Analyze class-wise area distribution and percentage changes
* Gain insights through intuitive visualizations

---

## ✨ Key Features

* 🗺️ **Interactive Map Visualization**
  Dynamically explore land-use data using GeoJSON layers

* 🔄 **Land-Use Transition Analysis**
  Track how different land categories evolve over time

* 📊 **Insightful Metrics Dashboard**

  * Total area change (~14.21 ha)
  * Class-wise percentage change
  * Major land transformation trends

* ⚡ **Responsive & Smooth UI**
  Designed for seamless interaction and real-time exploration

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Geospatial Libraries:** Leaflet.js, Turf.js
* **Data Format:** GeoJSON
* **Deployment:** Vercel

---

## 🧩 How It Works

1. Load GeoJSON datasets representing land-use for different years
2. Compare spatial layers to detect changes
3. Compute transitions between land-use classes
4. Visualize results through interactive maps and metrics
